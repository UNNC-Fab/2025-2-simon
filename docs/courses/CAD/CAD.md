# Week 4 - CAD Computer-aided design


## Fusion 360

Cloud based CAD tool for collab product development. Fusion 360 allow user to create detailed 3D models and test through simulations. Projects are also to be processed through 3D printing, all data is saved in cloud which enables team based project but also requires internet or ethernet to proceed. 

## Key features

1. Sketching: Create 2D sketch with line, circle, arc etc to create precise designs.

2. 3D Modeling: 
    * Parameters and extrusions to develop 3D model
    * Revolve to create complex shape

3. Assemble: Combine components together, define joint and simulate movements

4. Simulation and Analysis: 

5. CAM (Computer Aided Manufacture)

6. Rendering: Create photo images of design for presentation

7. Cloud Storage: Allow projects to be shared online and real time collab

8. Add in and Extension: Extend function for plugins.
